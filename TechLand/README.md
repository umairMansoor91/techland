# TechLand

A modern web application for TechLand - Resource Augmentation & Engineering Talent solutions.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linting
npm run lint
```
